Apache kafka

Jul 13, 2019 | tool

是什么
简单来说，Kafka是一个消息中间件。生产者产生消息，消费者消费消息。
适用场景

  1. 系统解耦

实战

在Ubuntu(Debian)下安装Apache Kafka。
Apache Kafka 依赖Java环境，所以需要先安装Java。

从官网下载Kafka压缩包，开箱即用。

Kafka使用ZooKeeper，先启动ZooKeeper服务,然后再启动Kafka服务。
